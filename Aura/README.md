# Aura — Your Care, Always Connected

A platform where doctors, patients, and families share one unified health record.

## Problem
Mental health patients lose medical files and forget their history — doctors and families are left without critical information.

## Solution
Aura gives all three users (Doctor, Patient, Family) access to the same secure health record — so nothing is ever lost.

## Features
- Real-Time Interaction
- Unified Profiles
- Doctor Discovery
